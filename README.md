# Open Terminal Here

Open the Terminal (Mac OS X, Linux) or Command Prompt (Windows)
in the given directory via context menu or keyboard shortcut.

Platform | Keyboard shortcut
-------- | -----------------
Mac OS X | `ctrl-cmd-t`
Windows  | `ctrl-shift-t`
Linux    | `ctrl-shift-t`
